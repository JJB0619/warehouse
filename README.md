# 餐饮仓库管理系统
一套适用于餐饮行业的多仓库库存管理系统，支持管理员/普通用户权限、密码加密、多仓库管理、出入库调拨、库存预警、Excel全报表导出。

## 功能特性
- ✅ 密码加密存储
- ✅ 管理员 + 普通用户双权限
- ✅ 多仓库管理：后厨 / 吧台 / 二楼 / 三楼
- ✅ 商品管理、入库、出库、仓库间调拨
- ✅ 库存预警自动检测
- ✅ 一键导出所有Excel报表
- ✅ 系统启动自动创建管理员
- ✅ 100% 适配 Railway 一键部署

## Railway 一键部署
1. Fork 本项目到你的GitHub
2. 在Railway中连接GitHub仓库
3. 自动部署完成，无需任何配置
4. 访问生成的域名即可使用

## 技术栈
Python Flask + SQLAlchemy + SQLite/PostgreSQL + Werkzeug密码加密 + OpenPyXL报表导出
